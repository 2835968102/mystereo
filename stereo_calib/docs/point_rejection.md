# 点剔除与过滤机制总结

整个标定流程中共有 **11 处** 过滤/剔除机制，按流水线顺序排列如下：

---

## 1. 特征匹配阶段（track_builder.cc）

### 1.1 匹配分数过滤

- **位置:** `track_builder.cc:229-237`
- **阈值:** `max_match_score`（默认 1.0）
- **作用:** 跳过 SuperPoint 匹配置信度低于阈值的匹配对

### 1.2 基础矩阵 RANSAC 几何过滤

- **位置:** `track_builder.cc:240-274`
- **阈值:** RANSAC 距离 2.0px，置信度 0.995
- **作用:** 用 `cv::findFundamentalMat` RANSAC 估计基础矩阵，标记不满足对极几何约束的匹配为非内点

### 1.3 内点数量/比例过滤

- **位置:** `track_builder.cc:254-266`
- **阈值:** `min_pair_inliers`（默认 12），`min_pair_inlier_ratio`（默认 0.35）
- **作用:** 整对图像的内点数或内点比例不达标则直接丢弃该图像对

### 1.4 Track 长度过滤

- **位置:** `track_builder.cc:389-391`
- **阈值:** `min_track_len`（默认 3）
- **作用:** 跨帧观测数不足的 track 被丢弃，保证每个 3D 点有足够约束

### 1.5 左右目观测完整性检查

- **位置:** `track_builder.cc:393-401`
- **作用:** 每条 track 必须同时包含左、右相机的观测，否则无法立体三角化，直接丢弃

### 1.6 像素位移过滤（帧间配准）

- **位置:** `track_builder.cc:541-557`
- **阈值:** `kMaxPixelDisp = 300.0` px
- **作用:** 帧间对应点的欧氏距离超过阈值时跳过，避免误匹配影响位姿初始化

---

## 2. 优化阶段（offline_stereo_ba.cc / stereo_factors.cc）

### 2.1 相机后方点惩罚（代价函数级别）

- **位置:** `stereo_factors.cc:37-42, 77-81, 129-134`
- **条件:** `Z <= 0`（点在相机后方）
- **作用:** 残差直接设为 `1e6`，通过大惩罚驱使优化器将点推到相机前方

### 2.2 活跃帧过滤（增量 BA）

- **位置:** `offline_stereo_ba.cc:62-90`
- **作用:** `RunBundleAdjustment` 中只纳入 `active_frames` 中已注册帧的观测；每条 track 至少需要 2 个活跃观测才参与优化

### 2.3 重投影误差外点剔除（BA 后迭代）

- **位置:** `offline_stereo_ba.cc:198-247`（`RejectOutliers`）
- **调用:** `offline_stereo_ba.cc:472-495`（迭代循环）
- **阈值:** `outlier_rejection_threshold`（默认 2.0px）
- **作用:** BA 收敛后计算每个观测的重投影误差，超过阈值的标记为 `rejected`；迭代执行（最多 `max_outlier_rejection_rounds` 轮，默认 100），每轮剔除后重新 BA，直到无外点可剔除

---

## 3. 最终校验阶段

### 3.1 FOV 合理性检查

- **位置:** `stereo_eval.cc:122-139`
- **阈值:** 10° ~ 160°
- **作用:** 检查优化后内参对应的 FOV 是否在物理合理范围内，如果 `fx <= 0`、`cx <= 0` 或 FOV 超出范围，判定优化发散

### 3.2 最终重投影误差阈值

- **位置:** `offline_stereo_ba.cc:499-507`
- **阈值:** `max_reproj_error`（默认 20.0px）
- **作用:** 最终全局 RMSE 超过该阈值则整次标定判定失败

---

## 流水线概览

```
SuperPoint 匹配
  ├─ ① 匹配分数过滤
  ├─ ② 基础矩阵 RANSAC
  ├─ ③ 内点数/比例检查
  │
  ▼
Track 构建
  ├─ ④ Track 长度过滤
  ├─ ⑤ 左右目完整性
  ├─ ⑥ 像素位移过滤
  │
  ▼
增量 BA
  ├─ ⑦ 相机后方点惩罚（代价函数）
  ├─ ⑧ 活跃帧过滤
  │
  ▼
最终全局 BA
  ├─ ⑨ 重投影误差迭代外点剔除
  │
  ▼
最终校验
  ├─ ⑩ FOV 合理性检查
  └─ ⑪ 最终 RMSE 阈值检查
```
